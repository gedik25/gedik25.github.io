İletişim

Bu bölümde CSS'in temel özelliklerini (display, width, height, margin, padding, border, selector) uygulayarak iki sütunlu bir sayfa düzeni oluşturulmuştur. Yalnızca HTML5 + CSS3 kullanılmıştır.

İçerik

Bu alanda örnek içerik metinleri bulunmaktadır. CSS özellikleriyle bu metinler arasında boşluklar (line-height) ve hizalamalar (text-align) düzenlenmektedir.

Önce CSS ile kutuları / alanları yan yana getirmeyi, ardından alt alta yerleştirmeyi öğrendik. Şimdi ise CSS seçicilerini kullanarak grup ve hiyerarşik seçimler yapmayı göreceğiz.

Yan Menü / Bilgi

Bu alan yan menü veya bilgi kısmıdır. CSS ile genişlik (width), yükseklik (height), kenar boşlukları (margin/padding) ve kenarlık (border) özelliklerini kullanarak düzenleyebiliriz.

Öğrendiklerimizi hiyerarşik ve grup seçicilerle uygulayıp farklı bölümlere stil verebiliriz. Kaynak koda GitHub repo üzerinden erişebilirsiniz.
