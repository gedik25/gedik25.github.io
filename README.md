Web Programlama Dersi Laboratuvar Ödevleri

## Proje Hakkında
GitHub Pages üzerinde yayınlanan kişisel tanıtım ve mini portfolyo sitesidir. Saf HTML5 + CSS3 + JavaScript ile geliştirilmiştir, herhangi bir framework kullanılmamıştır.

## Sayfalar
- **index.html** — Ana Sayfa (iki sütunlu layout, hızlı bağlantı kartları)
- **about.html** — Hakkımda (biyografi, zaman çizelgesi, blockquote)
- **interests.html** — İlgi Alanlarım (ul, ol, dl listeleri)
- **contact.html** — İletişim (e-posta, GitHub, LinkedIn, iletişim formu)

## Haftalık İlerleme

### Hafta 1 — GitHub Pages & HTML
- Repository oluşturma ve GitHub Pages yayını
- 4 sayfalık kişisel tanıtım sitesi (HTML yapısı)
- Semantik etiketler: header, nav, main, footer, section
- Navigasyon, listeler, blockquote, anchor bağlantıları

### Hafta 2 — CSS Temelleri
- `display: inline-block` ile yatay menü ve iki sütunlu layout
- Hiyerarşik seçiciler: `nav a {}`, `.main-content p {}`, `.sidebar p {}`
- Grup seçici: `h1, h2, h3 {}`
- Metin düzenleme: font-size, color, line-height, text-align
- Responsive tasarım (mobilde tek sütuna geçiş)
